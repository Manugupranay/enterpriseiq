———————# EnterpriseIQ

AI Finance & Sales Intelligence Agent - RAG + LLM + ML + MCP

## Overview

EnterpriseIQ is a production-grade AI agent built for finance and sales teams. It combines Retrieval-Augmented Generation (RAG), Large Language Models, Machine Learning pipelines, and MCP integrations to deliver real-time intelligence from enterprise data.

## Tech Stack

- Python - core application
- RAG - document retrieval and grounding for factual answers
- LLM (Claude / OpenAI) - reasoning and natural language generation
- ML - predictive models for sales forecasting and financial signals
- MCP - multi-context protocol for agent orchestration
- Railway - cloud deployment

## Features

- Ask natural-language questions over financial documents and CRM data
- Sales pipeline forecasting using ML models
- RAG-grounded answers with cited sources
- MCP-integrated multi-agent workflows
- REST API for frontend consumption

## Live

Deployed on Railway. See deployment dashboard for live endpoints.
